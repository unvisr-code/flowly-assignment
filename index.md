# 🧩 고객 인터뷰 준비

**주제:** 대학생 예비창업가 대상 고객 인터뷰 준비

**인터뷰 일정:** 2025년 11월 2주차 / 광진청년센터 입주 창업팀 대상

---

## 1️⃣ 인터뷰 목적

대학생 예비창업가들이 **데이터 기반 의사결정의 중요성은 인식하지만, 실제로 데이터를 어떻게 수집·활용해야 하는지 어려움을 느끼는 이유**를 파악하고자 한다.

또한, **AI 기반 데이터 분석 도구가 이들의 문제를 어떻게 해결할 수 있는지**를 검증한다.

---

## 2️⃣ 인터뷰 대상 고객 프로필

| 항목 | 내용 |
| --- | --- |
| **대상 유형** | 세종대학교 및 광진청년센터 입주 대학생 예비창업가 |
| **인원 수** | 총 3명 (각기 다른 업종/팀 구성) |
| **특징** | - MVP 또는 프로토타입을 보유하고 있음- 웹/앱 기반 서비스를 운영 또는 기획 중- 데이터 분석 경험은 부족하지만 필요성은 인지- IR 피칭, 창업 경진대회 경험이 있음 |
| **선정 이유** | 제품의 초기 핵심 타깃층으로, 실제 데이터 활용 니즈와 문제 인식이 가장 뚜렷함. |

---

## 3️⃣ 가설 및 인터뷰 질문 설계

### 가설 1️⃣

**“대학생 예비창업가는 데이터의 중요성을 인식하지만, 분석을 실행할 수 있는 역량이나 도구가 부족하다.”**

**검증 질문**

- 창업 아이템을 운영하면서 데이터를 추적하거나 분석해본 경험이 있나요?
- 데이터를 다뤄야 한다고 느꼈던 순간은 언제인가요?
- 데이터를 분석하지 못했던 가장 큰 이유는 무엇이었나요? (시간 / 기술 / 인력 / 비용 등)
- 구글 애널리틱스나 Notion, Excel 등 어떤 툴이라도 사용해본 적이 있나요?

---

### 가설 2️⃣

**“데이터 분석은 어렵고 복잡하다는 인식이 강하며, 직관적인 리포트 형태를 선호한다.”**

**검증 질문**

- ‘데이터 분석’이라는 말을 들었을 때 가장 먼저 떠오르는 것은 무엇인가요?
- 현재 자신의 서비스 성과(예: 방문자 수, 클릭률, 구매율 등)를 어떻게 확인하고 있나요?
- 데이터를 직접 분석해본 적이 있다면, 어떤 부분에서 가장 어려움을 느꼈나요?
- 복잡한 수치 대신 자동 리포트로 핵심 요약을 볼 수 있다면 도움이 될까요?

---

### 가설 3️⃣

**“데이터 시각화와 AI 요약 리포트는 대학생 창업가에게 ‘시간 절약’과 ‘피칭 설득력 강화’로 이어진다.”**

**검증 질문**

- IR 피칭이나 경진대회에서 데이터를 근거로 성과를 보여달라는 요구를 받은 적이 있나요?
- 피칭 시 데이터를 어떻게 준비하나요?
- 자동으로 시각화 리포트를 만들어주는 도구가 있다면, 어떤 상황에서 활용하고 싶나요?
- 투자자나 멘토에게 보여주기 좋은 형태의 리포트를 원하시나요, 아니면 내부 개선용 리포트를 원하시나요?

---

### 가설 4️⃣

**“AI 기반 데이터 분석 툴에 대한 흥미와 수용 의지가 높다.”**

**검증 질문**

- AI 기반으로 데이터를 분석해주는 서비스를 들어본 적이 있나요?
- 직접 데이터를 해석하지 않아도 결과를 제안해주는 AI 도구가 있다면 사용해볼 의향이 있나요?
- 어떤 조건(예: 무료 체험, 간단한 UI, 자동 리포트 제공 등)이라면 실제로 써볼 의향이 있나요?
- 유료 구독 전환의 기준은 무엇일 것 같나요?

---

## 4️⃣ 인터뷰 진행 방식

| 항목 | 내용 |
| --- | --- |
| **장소** | 광진청년센터 회의실 |
| **형태** | 인터뷰  |
| **시간** | 1인당 약 15분 (3명 총 1시간 내외) |
| **기록 방식** | 인터뷰 기록 + 음성 녹취 (사전 동의 후) |
| **질문 순서** | ① 창업 현황 → ② 데이터 경험 → ③ 문제 인식 → ④ 해결 기대 → ⑤ AI 솔루션 수용도 |

---

## 5️⃣ 기대되는 결과

| 검증 항목 | 예상 반응 | 인사이트 활용 |
| --- | --- | --- |
| 데이터 분석 경험 | 낮음, ‘어렵다’는 인식 높음 | 제품의 ‘쉬운 데이터 분석’ 메시지 강화 |
| 데이터 활용 의지 | 강함 (투자 피칭용 근거 필요) | AI 리포트 자동화 기능 우선 개발 |
| AI 도입 태도 | 긍정적 (시도 의향 높음) | 무료 체험형 온보딩 설계 필요 |
| 사용 의향 | 단기 체험 후 장기 구독 가능성 있음 | 2주 체험판 + 리마인드 메일 시나리오 설계 |
